## Project Overview

Bootable Linux USB toolkit that transforms a Microsoft Surface Pro 7 into a LeRobot SO-101 robot arm control station. The system automates everything from flashing the USB drive on Windows to fully configuring Ubuntu with the linux-surface kernel, Python environment, LeRobot v0.5.0, and servo hardware access -- using Claude Code as an AI-driven orchestrator for the post-install phases.

## Quick Reference

- **Tech Stack:** Bash, PowerShell 5.1, Python 3.12, Ubuntu 24.04.2 LTS, LeRobot v0.5.0
- **Entry Points:** `flash.ps1` (Windows), `setup.sh` (Linux), `CLAUDE.md` (AI orchestration)
- **Architecture Pattern:** Multi-stage provisioning pipeline (Windows prep -> Ubuntu install -> AI-driven configuration)
- **Target Hardware:** Surface Pro 7 (i5-1035G4, 8GB RAM, Intel Iris Plus)
- **Servo Hardware:** Feetech STS3215 servos via SO-101 arm kit (CH340 USB serial)

## Getting Started

### Prerequisites

- Windows PC with PowerShell 5.1+ (for USB flashing)
- 64GB+ USB 3.0 stick (SanDisk recommended)
- USB wireless keyboard with 2.4GHz dongle
- Multi-port USB adapter
- Surface Pro 7 with Secure Boot disabled

### Flash the USB (Windows)

```powershell
.\flash.ps1
```

### Install Ubuntu (Surface)

Follow [BOOT-GUIDE.md](./BOOT-GUIDE.md) for the toram install method.

### Configure (Surface, after Ubuntu installed)

```bash
sudo apt update && sudo apt install -y git
git clone https://github.com/Quintinity/linux-usb.git ~/linux-usb
cd ~/linux-usb
chmod +x setup.sh
./setup.sh
claude
# Say: "continue setup"
```
