## Production notes

- **Loop-friendly cut.** Last 0.5s fades to black; first frame opens on black. The loop boundary is invisible.
- **Mute-friendly.** Every visual beat is also a text overlay. Assume no one hears the VO.
- **Aspect + spec.** 16:9, 1920x1080, H.264. Target file 8 MB, ~1.1 Mbps. AAC audio at -16 LUFS.
- **Captions.** Burned-in white sans-serif with drop shadow. No separate VTT — LinkedIn prefers burned-in and the stand monitor has no caption track.
- **Recording schedule.** Capture footage during W2/W3 bench testing. Final cut by T-3 days (2026-05-22). One re-cut window between T-3 and T-1. Videographer: {{PLACEHOLDER: contracted videographer name + rate}}.
