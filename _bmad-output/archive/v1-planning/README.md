# Archived: v1 Planning Artifacts (Pre-Citizenry)

These documents are the original armOS planning artifacts from before the
citizenry protocol was built. They assumed a monolithic single-machine
architecture that has been superseded by the distributed citizenry model.

**Do not use these for current planning.** They are preserved for reference only.

Archived: 2026-03-16

Current planning artifacts are in `_bmad-output/planning-artifacts/` and
are prefixed with `citizenry-v2` or `citizenry-v3`.
